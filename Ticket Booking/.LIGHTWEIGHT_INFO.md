# 📦 Lightweight Project Setup

## What Was Removed?

To make this project lightweight and easy to share, the following folders have been removed:

### 1. `frontend/node_modules/` (~233 MB)
- Contains all Node.js packages for the React frontend
- Installed via `npm install`

### 2. `backend/target/` (~59 MB)
- Contains compiled Java classes and build artifacts
- Created by Maven during build process

### Total Space Saved: **~292 MB**
### Current Project Size: **776 KB**

---

## Why Remove These Folders?

✅ **Easy to Share:** Small file size makes it easy to upload, download, or share via email/USB
✅ **Clean Repository:** No bloated dependencies in version control
✅ **Fresh Setup:** Each developer can install exact versions needed
✅ **No Conflicts:** Prevents platform-specific binary conflicts

---

## How to Restore and Run?

### Step 1: Install Backend Dependencies

```bash
cd backend
mvn clean install -DskipTests
```

This will:
- Download all Java libraries from Maven Central
- Compile the project
- Create the `target/` folder (~59 MB)

---

### Step 2: Install Frontend Dependencies

```bash
cd frontend
npm install
```

This will:
- Download all npm packages from npm registry
- Create the `node_modules/` folder (~233 MB)
- Generate `package-lock.json`

---

### Step 3: Start the Application

**Backend (Terminal 1):**
```bash
cd backend
export JAVA_HOME="/path/to/your/java17"
mvn spring-boot:run -Dspring-boot.run.profiles=h2
```

**Frontend (Terminal 2):**
```bash
cd frontend
npm run dev
```

**Access:** http://localhost:3000

---

## What's in `.gitignore`?

The following are automatically ignored by Git:

```
# Backend
/backend/target/
/backend/.mvn/
/backend/*.iml
/backend/.idea/
/backend/data/

# Frontend
/frontend/node_modules/
/frontend/dist/
/frontend/.vite/
/frontend/package-lock.json

# Other
.DS_Store
*.log
```

This ensures these folders are never committed to version control.

## Quick Commands

### To Remove Heavy Folders Again
```bash
rm -rf frontend/node_modules backend/target
```

### To Install Everything at Once
```bash
# Backend
cd backend && mvn clean install -DskipTests

# Frontend
cd frontend && npm install
```

### To Check Current Size
```bash
du -sh .
```
